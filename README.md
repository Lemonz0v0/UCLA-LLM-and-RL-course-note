# UCLA-LLM-and-RL-course-note
本人是上海科技大学计算机硕士研0，方向safe RL/offline RL。目前正在学习和研究RL-LLM。UCLA的Ernest Ryuk教授在2025夏天开展的《LLM与强化学习》课程我认为非常前沿和细致，值得仔细学习，课程总时长11小时，总结出此笔记。
笔记共包含三个章节：RL、LLM、RLHF。RL章节总结了从TRPO到GRPO的具体细节和表示方法，其中还扩展了AlphaGo的具体细节；LLM章节总结了LM的组成形式，介绍从RNN到GPT-1再到T5的发展与具体细节，其中扩展了Transformer的具体细节；RLHF章节较为前沿，划分的较为杂乱，包含了例如IstructGPT这种LLM、最佳n采样这种模块和RLVR等内容，可以选择性阅读。
笔记内容90%为Ernest Ryuk教授在课程的原话，另外10%是个人的思考和总结，以及图片的扩充。RL部分较为潦草因本人之前已经学过。
